HTML Element

The HTML element is everything from the start tag to the end tag

<tagname> Content here... </tagname>

Root element

root element defines the whole HTML document.start with tag <html> and end with tag </html> then inside root element is <body> element.

Example

<html> . . . </html>

Body element

body element will defines the doccument's body. it start with tag <body> and end with tag </body> then inside body element there are two other element <h1> and <p>.

Example

<html> <body> . . . </body> </html>

Heading element

The <h1> element defines a heading.it start with tag <h1> and an end with tag </h1> and we can change number behide character h for change front size

Example

<html> <body> <h1> -text- </h1> . . </body> </html>

Solution

Paragraph element

Example

Solution

HTML Attribute

-All HTML elements can have attributes
-Attributes provide additional information about elements
-Attributes are always specified in the start tag
-Attributes usually come in name/value pairs like: name="value"

The href Attribute

The <a> tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to

Example

<html> <body> <a href="https://kantinun.github.io/">Visit my website</a> . . </body> </html>
