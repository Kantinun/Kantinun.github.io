HTML Element

Root element

root element defines the whole HTML document.start with tag <html> and end with tag </html> then inside root element is <body> element.

Example

<html> . . . </html>

Body element

body element will defines the doccument's body. it start with tag <body> and end with tag </body> then inside body element there are two other element <h1> and <p>.

Example

<html> <body> . . . </body> </html>

Heading element

The <h1> element defines a heading.It start with tag <h1> and an end with tag </h1>

Example

<html> <body> <h1> -text- </h1> . . </body> </html>
