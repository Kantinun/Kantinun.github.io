HTML Element

Root element

root element defines the whole HTML document.start with tag <html> and end with tag </html> then inside root element is body element.

Example

<html> . . . </html>

Body element

body element will defines the doccument's body. it start with tag <body> and end with tag </body> then inside body element there are two other element heading element and paragraph element.

Example

<html> <body> . . . </body> </html>
